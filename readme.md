# 📘 Procrastination Risk Predictor (AI/ML Project)

## 🔍 Overview

The Procrastination Risk Predictor is a Machine Learning project that
analyzes a student's daily habits and predicts whether they are likely
to procrastinate.

## 🎯 Problem Statement

Procrastination is a common issue among students that leads to poor
academic performance, stress, and inefficient time management.

## 💡 Solution

This project uses a Machine Learning classification model to analyze
behavior and predict procrastination risk.

## ⚙️ Features

-   Behavioral data analysis
-   Machine learning prediction
-   Real-time user input
-   Model evaluation

## 🧪 Technologies Used

-   Python
-   Pandas
-   Scikit-learn

## 🧠 Machine Learning Approach

Model: Logistic Regression

### Inputs:

-   Study hours
-   Screen time
-   Breaks
-   Sleep hours
-   Deadline days

### Output:

-   0 → Not Procrastinating
-   1 → Procrastinating
